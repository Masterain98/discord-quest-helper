## What's New

### 🏗 Performance & Architecture
- **Embedded Runner Binary**: Runner is now compiled directly into the application, eliminating runtime path resolution failures and improving startup reliability.
- **Enhanced Runner Visibility**: Runner build metadata (commit hash, build time, size) is now exposed and displayed in the Debug UI for better transparency.
- **Centralized Request Builder**: Unified HTTP request handling for more consistent and maintainable API interactions.

### ⚡ Reliability
- **Improved Resource Management**: Runner executable location now uses AppHandle for better path resolution across different deployment scenarios.
- **Updated Dependencies**: Bumped all dependencies to latest versions for improved security and performance.

### 🎨 User Experience
- **UX Optimization**: Refined user interface interactions and improved overall responsiveness.
- **Better Debug Information**: Runner information panel in Debug view shows embedded status, version, and build details.
